###考察点
双指针

###描述
给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。
不要使用额外的数组空间，你必须在原地修改输入数组并在使用O(1)额外空间的条件下完成。

###思路
* 用到两个指针
* 从数组中的第二个数字开始和前一个数字进行比较，不相等，则跳过继续
* 相等i不做增加，等同于删除


###代码
public int removeDuplicates(int[] nums) {
    if (nums.length == 0) 
    return 0;
    int i = 0;
    for (int j = 1; j < nums.length; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}
